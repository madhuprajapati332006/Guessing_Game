# 🎯 Guessing Game

A simple Python-based word guessing game where the player chooses a difficulty level and tries to guess a randomly selected secret word. After each incorrect guess, the game provides a hint by revealing correctly matched letters.

## 🚀 Features
- Three difficulty levels: Easy, Medium, Hard
- Random word selection
- Hint system for incorrect guesses
- Tracks the number of attempts
- Beginner-friendly Python project

## 🛠️ Technologies Used
- Python 3
- Random module

## ▶️ How to Run

1. Clone the repository:
bash
git clone https://github.com/your-username/guessing-game.git


2. Navigate to the project folder:
bash
cd guessing-game


3. Run the program:
bash
python guessing_game.py
